# RC12.6 Alcohol and Operating Profit Fix

- Alcohol sales now prefers Toast alcohol category rows when available.
- Toast overall alcohol sales is no longer added to beer/liquor/wine/margarita component fields.
- Margaritas and cocktails remain included in Alcohol Sales.
- Added Beer, Wine, Liquor, and Margarita sales breakdown values to the department engine.
- Operating Profit now uses the direct accounting formula:
  Toast Net Sales - Operating Payroll (tips excluded) - Vendor/Business Spend.
- Operating Profit no longer changes because of food/alcohol department allocation percentages.
- Restaurant Health popup no longer displays meaningless $0 subtotal/clicked-total rows for percentage metrics.
