# RC3.1.2 Separate Tips and Kitchen Payroll Tabs

## Changes
- Added a dedicated **Tips Payroll** review tab.
- Added a dedicated **Kitchen Payroll** review tab.
- Kept an **All Pending** tab for combined review.
- Tips Payroll includes employees with Toast tips or tipped job roles such as server, waiter, bartender, and front-of-house.
- Kitchen Payroll includes the remaining non-tipped operating labor rows.
- Each tab supports select-all for only the rows visible in that tab.
- Approve Selected continues to approve only checked rows.
- Existing Tips History and All Payroll views remain available.
- Payroll now opens on Tips Payroll by default.

## Verification
- Production build completed successfully with Vite.
