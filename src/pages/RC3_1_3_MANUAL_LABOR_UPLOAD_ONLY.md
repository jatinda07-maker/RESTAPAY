# RC3.1.3 Manual Labor Upload Only

Payroll no longer connects to or refreshes from the automatic Toast labor sync.

## Changes

- Removed automatic `toast_labor` loading from the Payroll page.
- Removed Refresh Toast Labor, pending Toast review, and automatic approval tabs.
- Payroll now opens on All Payroll.
- Kept Import Labor Summary for CSV/XLS/XLSX uploads.
- Kept Manual Payroll and Payroll Groups as separate tabs.
- Renamed the source filter to Imported Labor Summary.

Sales and other Toast automation are not changed by this patch.
