Resta Pay Gemini Fix

1. Do NOT open index.html directly.
2. Set your Gemini API key:
   set GEMINI_API_KEY=your_key_here
3. Run Start Restaurant App.bat, or run:
   npm start
4. Open http://localhost:4173
5. Go to Invoices > AI upload / AI camera > Read With AI.

If you see Failed to fetch, the browser cannot reach the local Node server at http://localhost:4173.
